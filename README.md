# bgrm : bg-remove

A CLI tool to image processing. This tool simply allows you to efficiently remove the background from an image without needing an Internet connection after it has been installed.

| Settings             | Description                                                                |
| ----------------- | ------------------------------------------------------------------ |
| -h / --help | Show this help message and exit. |
| -v / --version | The param "Version". Use to know bgrm version. |
| -i / --image | The param "Image". Use to remove image background. |
| -g / --get | The param "Get". Use to get user history (logs) about bgrm. |
| -c / --clean | The param "Clean". Use to clean user history(logs). |
